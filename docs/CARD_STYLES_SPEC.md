# 四种卡片风格设计规范

> 对应设计规范：Traditional ink-wash · Minimalist · Modern oriental · Festive scroll

## 设计要素总览

| 要素 | 规范 |
|------|------|
| **风格** | 传统水墨感、极简、现代东方美学 |
| **构图** | 竖版海报、对称、中心卷轴/色块 |
| **色彩** | 喜庆深红、金黄、水墨黑 |
| **质感** | 复古纸张纹理、宣纸感 |
| **细节** | 奔马剪影、水墨飞溅、中国书法 |

---

## 1. 传统水墨 (inkWash)

| 维度 | 实现 |
|------|------|
| **风格** | Traditional Chinese ink-wash style |
| **背景** | 喜庆深红渐变 + 复古纸张纹理 + 水墨飞溅 |
| **中心** | 金色卷轴 (GoldenScroll)，米色内底 |
| **主图** | 奔马剪影 (GallopingHorse)，水墨黑，柔化边缘 |
| **文字** | 中国书法字体，金黄主标题，水墨黑正文 |
| **装饰** | 背景「马」字淡入、除夕守岁·辞旧迎新 |

---

## 2. 极简 (minimal)

| 维度 | 实现 |
|------|------|
| **风格** | Minimalist |
| **背景** | 宣纸色 #FFF8E7，细边框 |
| **中心** | 细线框卷轴，白底，大留白 |
| **主图** | 无马，纯文字 |
| **文字** | 水墨黑，克制字号，细线分隔 |
| **装饰** | 顶部/底部极细横线，年份居中 |

---

## 3. 现代东方 (modern)

| 维度 | 实现 |
|------|------|
| **风格** | Modern oriental aesthetic |
| **背景** | 深红渐变 + 金色粗边框 + 纸张纹理 |
| **中心** | 现代感卷轴，米白渐变内底，几何线条 |
| **主图** | 奔马剪影（无水墨柔化），线条清晰 |
| **文字** | 水墨黑，几何分隔线 |
| **装饰** | 顶部渐变线 + 年份，简洁对称 |

---

## 4. 喜庆卷轴 (festive)

| 维度 | 实现 |
|------|------|
| **风格** | 喜庆、对称、书法 |
| **背景** | 鲜红渐变 + 复古纹理 |
| **中心** | 金色卷轴，米色内底 |
| **主图** | 梅花三朵（替代马） |
| **文字** | 金黄/深金书法 |
| **装饰** | 四角梅花，除夕守岁·辞旧迎新·年份 |

---

## 技术实现

- `CardTemplate`: `inkWash` | `minimal` | `modern` | `festive`
- 共用组件：`VintageTexture`、`GoldenScroll`、`GallopingHorse`、`PlumBlossom`、`InkSplashes`
- 色彩：`PALETTE.red`、`PALETTE.gold`、`PALETTE.ink`、`PALETTE.rice`
