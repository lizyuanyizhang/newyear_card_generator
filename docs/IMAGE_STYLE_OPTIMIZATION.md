# 贺卡图片风格优化方案

> 参考提示词：Minimalist Chinese Lunar New Year vertical poster, festive deep red + vintage paper texture, golden scroll, galloping horse in Shuimo ink-wash style, traditional oriental aesthetic

## 一、对照分析

| 提示词元素 | 当前实现 | 优化方向 |
|------------|----------|----------|
| **背景** | 纯红渐变 | 添加 subtle vintage paper texture（复古纸张纹理） |
| **主体框架** | 无独立框架 | 添加 golden-yellow rectangular scroll 卷轴/画框 |
| **马图案** | 金色剪影、静态、边缘装饰 | 水墨风格、奔跑态、置于卷轴内、黑色/深灰墨色 |
| **版式** | 内容直接铺在红底上 | 内容置于金色卷轴内部，层次更清晰 |
| **文字层级** | 较扁平 | 顶部主标题 / 中部祝福 / 底部小字（除夕守岁、年份等） |
| **整体气质** | 喜庆、偏平 | 东方传统、克制、clean composition |

---

## 二、具体优化措施

### 1. 背景：复古纸张纹理

- **实现**：CSS `background-image` 叠加低透明度噪点/纹理图层
- **效果**：红色背景上增加 subtle 颗粒感，类似宣纸/老纸
- **技术**：SVG 噪点 filter 或 base64 纹理图，`mix-blend-mode: overlay`，opacity 5–15%

### 2. 金色卷轴 / 画框

- **结构**：居中垂直矩形，金色描边 + 浅金/米色内底
- **尺寸**：约占画面 70–80% 宽度，留出红边
- **风格**：类似竖轴书法卷，顶部可加卷轴轴头装饰（可选）

### 3. 马意象：水墨风格

- **颜色**：由金色改为深灰/墨黑 `#1a1a1a`，半透明模拟水墨
- **动态感**：使用奔跑/跃起姿态的马剪影（可替换为 galloping horse SVG）
- **位置**：移至卷轴内部，作为主视觉之一，与文字平衡

### 4. 版式结构（卷轴内）

```
┌─────────────────────┐
│    [主标题·金]       │  新春快乐 / 驹影映福
│  ─────────────────  │
│   [水墨马·居中偏上]   │  动态奔跑马剪影
│                     │
│   致 [收件人]        │
│   [祝福语]           │  主文案
│                     │
│   除夕守岁·辞旧迎新   │  小字装饰
│   [年份]             │
│   —— [落款] 敬上     │
└─────────────────────┘
```

### 5. 配色微调

- **红底**：保持 `#E41A00` → `#C41600` 渐变，可略加深为更「深红」
- **卷轴**：描边 `#D8B038`，内底 `rgba(255,248,220,0.95)` 或类似
- **马**：`#1a1a1a` 或 `#2d2d2d`，opacity 0.7–0.9
- **小字**：黑 `#1a1a1a` 与金 `#D8B038` 混排

## 三、四风格差异化

| 风格 | 卷轴 | 马 | 纹理 | 额外 |
|------|------|-----|------|------|
| **马意象** | ✅ 金色卷轴 | ✅ 水墨奔跑马 | ✅ 纸张纹理 | 马字装饰保留 |
| **梅花** | ✅ 金色卷轴 | ❌ 无马 | ✅ 纸张纹理 | 梅花角标 |
| **卡通** | 圆角浅金框 | 🐴 emoji 或 Q 版马 | 轻纹理/无 | 保持活泼 |
| **极简** | 细线框 / 无框 | ❌ | 无 / 极轻 | 留白为主 |
